# Steps to Create the Same Code Structure

1. Install Node.js and npm if not already installed
2. Run 'npx create-next-app@latest project-name --typescript --tailwind --eslint --app' to create a new Next.js project
3. Navigate to the project directory: 'cd project-name'
4. Create .vscode directory and add settings.json
5. Create app/api/approvals/ directory and add route.ts
6. Create app/approvals/ directory and add page.tsx
7. Add favicon.ico, globals.css, layout.tsx, page.tsx to app/ directory
8. Add SVG files (file.svg, globe.svg, next.svg, vercel.svg, window.svg) to public/ directory
9. Add root files: .gitignore, eslint.config.mjs, next.config.ts, package-lock.json, postcss.config.mjs, README.md, tsconfig.json
